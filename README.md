#### 구현 기능
- `mouse event`를 이용한 `canvas drawing`
- `a 태그`의 `href` 및 `download` 속성을 이용한 이미지 저장
- `canvas` 속성을 이용한 `canvas` 채우기
- 색상(div)을 눌렀을 때 `range bar(accentColor)` 색상 변경 및 `strokeStyle` 변경
- `range` 를 통한 `line width` 조절
